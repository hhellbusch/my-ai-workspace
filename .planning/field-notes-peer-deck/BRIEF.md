## Deck type

**Master reference deck** — structured like a book, not a 20-minute talk.

~88 slides is appropriate because the deck is navigated in chapters, not read front-to-back in one sitting. A live presenter covers one or two acts; an async reader starts where it's relevant.

This is not a lightning talk, a step-by-step tutorial, or an interactive workshop.

---

## Audience

**Mixed technical peers** — some close (colleagues, team), some distant (peer network).

Delivery may be live (presenter picks acts) or async (reader navigates independently). The deck must work for both without requiring a facilitator to explain how to use it.

Assumed background: software/systems practitioner. Not assumed: prior exposure to AI tooling, agent frameworks, or PKM systems.

---

## Goals

Four goals, all real, weighted:

| # | Goal | Weight |
|---|---|---|
| 1 | **Intentionality** — peer starts using AI tools deliberately, not just as a chat box | Primary |
| 2 | **Harness mental model** — understands harness > model; knows how to build context deliberately | Primary |
| 3 | **Epistemics** — treats AI output as unvalidated claims; verification posture internalized | Strong secondary |
| 4 | **Access** — knows the repo exists, what's in it, where to start | Supporting |

---

## Navigation principle

**Acts are self-contained chapters.** A reader can start at any act and get value without the prior ones. Act I carries the core thesis; later acts deepen it.

Suggested paths for different audiences:
- Just want the harness model → **Act I + Appendix**
- Building an AI workspace → **Acts I + V**
- Full learning path → **All five acts**
- Already using AI tools, want verification posture → **Acts I (mechanism block) + IV**

The appendix is the go-deeper reading map — each entry has a one-line *why read*.

---

## Out of scope

- Step-by-step tutorial or lab exercise
- Interactive workshop format
- Live demo (slides reference library entries and case studies; no live tooling)
- Introductory "what is AI?" content — assumes practitioner background
- Biographical / career narrative about the author
